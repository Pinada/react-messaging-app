### Built With

* [React](https://reactjs.org/)



### Installation


1. Clone the repo
   ```sh
   git clone https://github.com/Pinada/react-youtube-watch-time-gate.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run
   ```sh
   npm start
   ```
